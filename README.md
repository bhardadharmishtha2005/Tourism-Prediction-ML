# Tourism-Prediction-ML
An end-to-end Data Science project predicting global travel patterns and user ratings using Random Forest and Hyperparameter Tuning.

### Technical Implementation
* **Data:** Merged 8 relational tables (Transactions, Cities, Countries, etc.).
* **EDA:** 8 professional visualizations including Correlation Heatmaps and ANOVA testing.
* **ML Model:** Random Forest (Classification & Regression).
* **Optimization:** RandomizedSearchCV used for hyperparameter tuning.

### Files in Repository
* `tourism_analysis.ipynb`: Full data analysis and model training.
* `tourism_clf_model.pkl`: The optimized trained model for deployment.
* `requirements.txt`: Necessary libraries for running the project.
